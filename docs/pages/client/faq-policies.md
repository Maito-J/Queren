# /faq - Frequently Asked Questions

## Purpose
Answer common questions to reduce support burden and build trust.

## UI Sections
1. **Search Bar** (optional for MVP)
2. **Category Tabs**: Booking, Pricing, Service, Account
3. **Accordion FAQ Items**

### FAQ Content (Initial)
**Booking**
- How do I book a cleaning?
- Can I reschedule or cancel?
- How far in advance should I book?

**Pricing**
- How is the price calculated?
- Are there any hidden fees?
- Do you charge for parking/supplies?

**Service**
- What's included in a regular cleaning?
- Do I need to be home during cleaning?
- What if something is damaged?

**Account**
- How do I update my address?
- Can I have multiple addresses?
- How do I leave a review?

## Acceptance Criteria
- [ ] Accordions expand/collapse smoothly
- [ ] Categories filter questions (if implemented)
- [ ] Mobile friendly
